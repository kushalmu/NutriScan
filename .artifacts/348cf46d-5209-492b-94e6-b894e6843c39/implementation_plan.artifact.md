# Fix Room Compilation Error (Kotlin 2.2.10 Metadata Mismatch)

The project is failing to build because Room 2.6.1's compiler (using Kapt) does not support the Kotlin 2.2.0 metadata generated by Kotlin 2.2.10.

## Proposed Changes

### [Component Name] Build Configuration

#### [MODIFY] [libs.versions.toml](file:///C:/Users/gunam/.gemini/antigravity-ide/scratch/NutriScan/gradle/libs.versions.toml)
- Update `room` version from `2.6.1` to `2.8.4`.
- Add the `ksp` plugin definition under `[plugins]`.

#### [MODIFY] [build.gradle.kts](file:///C:/Users/gunam/.gemini/antigravity-ide/scratch/NutriScan/build.gradle.kts) (Root)
- Add `alias(libs.plugins.ksp) apply false` to the plugins block.

#### [MODIFY] [build.gradle.kts](file:///C:/Users/gunam/.gemini/antigravity-ide/scratch/NutriScan/app/build.gradle.kts) (App)
- Remove `id("kotlin-kapt")`.
- Add `alias(libs.plugins.ksp)`.
- Replace `kapt(libs.room.compiler)` with `ksp(libs.room.compiler)`.
